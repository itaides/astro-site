# Site Vision: Arctica Digital

## 1. Vision

Arctica Digital is a global branding and digital design agency building products, services, and eCommerce experiences. The website should position Arctica as a **premium, strategy-led creative studio** — not a generic agency. The site communicates confidence, technical expertise, and a collaborative partnership approach.

**Target Audience:** Marketing Directors, CMOs, Founders, and Product Managers at mid-to-large businesses and scaling startups across healthcare, e-commerce, non-profits, legal, food & beverage, and technology.

## 2. Tech Stack

- **Framework:** Astro 5.x (static site generation)
- **Hosting:** Cloudflare Pages
- **Styling:** Vanilla CSS with custom design tokens
- **Fonts:** Outfit + Inter (Google Fonts)
- **Animations:** CSS animations + Intersection Observer
